#утвердительное
##present
- кто + am/is/are + какой/кто
- I am ready

##perfect
- кто + have/has been + какой/кто
- I have been ready

#отрицательное
##present
- кто + am/is/are + not + какой/кто
- I am not ready

##perfect
- кто + have/has not been + какой/кто
- I have not been ready

#вопросительное
##present
- am/is/are + кто + ... ?
- Am I ready?

##perfect
- have/has + кто + been + ... ?
- Have I been ready?

#восклицательное
##present
- How + какой + кто + is/are!
- How ready I am!

##perfect
- How + какой + кто + have/has been!
- How ready I have been!
